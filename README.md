# Organizer

Python script what organize you files into folders.

## Install

Install the Organizer

```bash
  git clone https://github.com/luca4hh/Organizer.git
  cd Organizer
```
    
## Use

Use the Organizer

#### Run the install script
```bash
    python3 installer.py
```
#### Run the script
```bash
    python3 organizer.py
```
